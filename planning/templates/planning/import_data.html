📥 Import Spreadsheet

This page imports data from the eScience Lab workforce planning Excel spreadsheet.

The import is idempotent — running it multiple times will not create duplicate records. It uses Django's get_or_create() to only add new data.

Before importing:

  1. Make sure the spreadsheet file is accessible on the server
  2. Set the SPREADSHEET_PATH environment variable to the file path, or place the file in the project root directory
  3. Or run from the command line: python manage.py import_spreadsheet --file=/path/to/file.xlsx
